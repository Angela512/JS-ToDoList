# To-Do List

바닐라JS를 사용한 일정관리 크롬 앱입니다.

## 구현한 기능 설명

* 실시간으로 시간 확인

* 사용자 이름을 입력받아서 기억

* 오늘 할 일을 추가, 저장하고 삭제

* 사용자의 위치에 따른 날씨 정보와 현재 기온, 최고 기온 그리고 최저 기온 확인

* 새로고침 시 배경화면 랜덤으로 변경

* 새로고침 시 하단에 명언 랜덤으로 변경
